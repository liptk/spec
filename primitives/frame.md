# Frame

A frame is a container for any primitive (except itself), and each primitive has it's own unique frame ID.

## Parts

- The primitive's [FrameId](#frame-ids)
- The primitive itself

## Binary encoding

| length          | parameter  |
| --------------- | ---------- |
| 1 byte          | FrameId    |
| remaining bytes | primitive  |
